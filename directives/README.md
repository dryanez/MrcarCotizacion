# Directives

This directory contains SOPs (Standard Operating Procedures) written in Markdown.

Each directive should define:
- **Goal**: What the directive accomplishes
- **Inputs**: What data/parameters are required
- **Tools/Scripts**: Which execution scripts to use
- **Outputs**: What the directive produces
- **Edge Cases**: Known issues, API limits, timing constraints

## Template

```markdown
# [Directive Name]

## Goal
What this directive accomplishes

## Inputs
- Input 1: Description
- Input 2: Description

## Tools/Scripts
- `execution/script_name.py`: What it does

## Process
1. Step 1
2. Step 2
3. Step 3

## Outputs
- Output 1: Description
- Output 2: Description

## Edge Cases
- Known issue 1
- API limit considerations
- Timing constraints
```
